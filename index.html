Popular Commands

Sento is fully updated to support modern Discord Slash Commands. Explore some of the core features below.

/ban user reason Admin

Bans a specified user from the server and records the incident in the moderation log history.

/warn user reason Mod

Issues a formal warning to a user, storing the tracking case details in their user profile card.

/rank [user] Utility

Displays the current leveling card, showing total XP progress, server rank, and message counts.

/todo action Utility

Manages a personal to-do list embedded within your direct messages or targeted text channels.

/stats General

Outputs performance stats, including cluster ping latency, active servers, and total member counts.

/help [command] General

Fetches the comprehensive command dictionary layout or details regarding a single specified tool.
