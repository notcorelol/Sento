Popular Commands

Sento runs completely on modern, snappy Slash Commands. Check out the setup tools.

/ban user reason Admin

Bans an unwanted user and saves the action directly to the moderator's case dashboard file.

/warn user reason Mod

Issues an automated warning ticket to a troublesome user and logs their moderation count history.

/rank [user] Utility

Generates a neat graphic showing current active server XP standings, levels, and rank counts.

/todo action Utility

Creates an interactive, personal checklist directly inside the channel or your secret DMs.

/stats General

Pings core processes and updates live cluster latency counts, server numbers, and shard counts.

/help [command] General

Brings up an interactive, sliding menu containing your specific configured modules guide.
