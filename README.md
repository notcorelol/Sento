# Sento 

A powerful, feature-rich Discord assistant built to enhance your server experience with advanced moderation, leveling systems, utility options, and entertainment modules.

## 🌐 Live Website
Check out our official web landing page here:
👉 **[https://notcorelol.github.io/sento](https://notcorelol.github.io/sento)**

## ✨ Features
* **Modern Slash Commands:** Built entirely around clean, native Discord interactions.
* **Advanced Moderation:** Automated moderation tools, logging channels, and safety tracking.
* **Leveling & Engagement:** Dynamic text-based XP tracking with auto-assigned rank roles.
* **Dynamic Theme Design:** Our documentation page supports auto-detection for system dark/light modes.
